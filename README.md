# Metro Launcher

**Metro Launcher** is a highly customizable, Windows Phone-inspired launcher for Android. It brings the iconic "Live Tile" interface to modern Android devices, featuring a smooth, responsive, and nostalgic user experience.

- **Live Tiles**: Experience dynamic tiles that flip and update, just like the original Windows Phone.
- **Pixel-Perfect Layout**: Tiles are rigorously aligned to a grid system, supporting 1x1 (Small), 2x2 (Medium), and 4x2 (Wide) sizes.
- **Icon Centering**: App icons are mathematically centered and scaled to exactly 50% of the tile size for a uniform, premium look.
- **Animations**: Includes the classic "Turn" exit animation when launching apps and smooth tile flip animations.

- **Persistence**: Your layout preferences are automatically saved and restored.
